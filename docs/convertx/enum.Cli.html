Bytes

Convert byte values (e.g., bytes to MB or human readable).

Fields

§num: u64

Number of bytes to convert.

§megabytes: bool

Convert bytes to megabytes.

§human_readable: bool

Convert bytes to a human-readable string (e.g., “1.00 MB”).
